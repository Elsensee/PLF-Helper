Preislistenpfleger Helper

Ein Tool für Benutzer von www.preislistenpfleger.de

Zum Projekt auf GitHub

Über

Im Jahre 2011 hatte der Benutzer "Wurzel Manager" die Idee eines solchen Tools und erstellte es in einem Script für OpenOffice/LibreOffice und Microsoft Office. Allerdings war es ziemlich beschränkt auf das automatische Auslesen der Preise aus der Zwischenablage und dem nachfolgenden einfachen Sortieren, sodass die freiwilligen Preislistenpfleger es einfach bloß kopieren mussten. Zu seiner Seite: Preislisten-Pfleger Tool

Ich fand die Idee klasse und fing an, eine erste Version eines Programms zu schreiben, das unabhängig von OpenOffice/LibreOffice oder Microsoft Office lief und die eingegebenen Preise selbstständig hochlädt. Nach einigen Problemen hatte ich endlich die Version 0.6.0.0 (warum ich dort mit der Zählung anfing, weiß ich nicht :D) nach 7-9 Wochen fertig. Aber Version 0.6.0.0 funktionierte nicht wirklich, also benötigte ich ein paar weitere Wochen um Version 0.7.0.0 fertigzustellen, die funktionierte. Endlich. Größtenteils.

Mist.

Danach wurden die Pausen zwischen den Versionen länger und länger.

Im August 2013 habe ich dann dieses GitHub Repositroy erstellt, um das Programm vollständig neuzuschreiben. (Außerdem würde ich das gerne für Linux machen - dank dem Mono-Projekt) Vor ein paar Wochen dann (März 2014) habe ich die Arbeit wieder aufgenommen und hier... bin ich! :D
